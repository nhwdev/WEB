선언 함수

  • 함수의 이름이 있다
  • 함수의 정의와 호출로 구분함
  • 호이스팅 대상임

익명 함수 : 이름이 없는 함수

변수 | 속성 = function([매개변수, ...]){
함수 호출시 실행할 소스 코드들
}
특정 변수나 속성에 대입되는 함수 구현시 주로 사용됨
주로 이벤트 핸들러 작성시 많이 사용

  • 이름이 없는 함수
  • 이벤트 처리용으로 변수에 익명함수를 저장해서 사용함
  • 호이스팅 대상이 아님
  • 함수 정의 방식
    1. 일반 익명함수
      let val1 = function([매개변수...]) {...}
    2. 화살표(=>) 함수
      let val1 = ([매개변수...]) => {...}

함수의 리턴

  1. 리턴 타입의 선언이 필요 없다. 자료형에 상관없이 반환이 가능함
  2. return 키워드(예약어)로 함수를 종료함. 함수의 끝(})을 실행하면 함수가 종료
  3. 값이 반환하는 경우 : return 필수
    1. return 값
    2. return 리턴이 있는 함수()
